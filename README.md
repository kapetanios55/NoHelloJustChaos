# NoHelloJustChaos

A satirical guide to workplace communication gone wrong. 

Visit the site: [https://kapetanios55.github.io/NoHelloJustChaos/](https://kapetanios55.github.io/NoHelloJustChaos/)

## What is this?

A parody of the "nohello" movement that takes things to the absolute extreme. This site demonstrates what happens when you skip the pleasantries AND all common sense.

## Disclaimer

This is pure satire. Please be kind and communicative with your coworkers. Say hello, provide context, and don't deploy to prod without testing.
